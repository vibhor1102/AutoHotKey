# AutoHotKey Scripts

Personal AutoHotkey v2 scripts and helper libraries.

## Third-party code

`UIA.ahk` and `UIA_Browser.ahk` are from Descolada's UIA-v2 project, licensed under the MIT License. See `THIRD_PARTY_NOTICES.md`.

## Local setup

1. Install AutoHotkey v2.
2. Copy `LocalConfig.example.ahk` to `LocalConfig.ahk`.
3. Fill in local-only values in `LocalConfig.ahk`.
4. Run `Multi-Script.ahk`.

`LocalConfig.ahk` is ignored by Git so personal shortcuts and passwords are not published.
